# componetizer-cli

CLI tool for initialize projects with different templates

## Installation

```bash
npm install webkit-cli -g
```

## Usage

Initialize a new project

```
webkit new
```

Know the version

```
webkit --version
```

Show list of available commands

```
webkit help
```
