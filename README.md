# componetizer-cli

CLI tool for initialize react projects with different templates

## Installation

```bash
npm install componetizer-cli -g
```

## Usage

Initialize a new project

```
componetizer new
```

Know the version

```
componetizer --version
```

Show list of available commands

```
componetizer help
```
